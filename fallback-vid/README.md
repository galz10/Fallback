# Fallback Video Package

This folder contains the production brief, storyboard, brand assets, and raw demo captures for a Fallback launch/product video inspired by the MagicPath storyboard.

## Files

- `questions-and-answers.md` - direct answers to the 13 requested production questions.
- `storyboard.md` - a Fallback-specific 90 second storyboard with scene timing, visual direction, on-screen text, and demo footage mapping.
- `demo-capture-log.md` - manual demo recordings and stills captured from the local Electron app.
- `assets/` - copied Fallback logo and mark PNGs from `src/renderer/assets`.
- `demos/raw/` - raw `.mp4` demo recordings and reference stills.

## Positioning

Fallback is a local-first desktop workbench that keeps GitHub context on your machine and connects it to the local repo state you are about to change. The video should feel like a calm, high-trust developer tool: dark UI, precise motion, source-control evidence, and confident recovery paths.

Primary line:

> GitHub context, saved where you work.

Alternate short line:

> A local home for the GitHub context your repo work depends on.

Release language:

> Public alpha
