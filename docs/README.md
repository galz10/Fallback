# Fallback Docs

Fallback is a local-first companion for GitHub work. It keeps the repository context you rely on every day, such as pull requests, issues, comments, checks, commit history, and local changes, close at hand on your computer.

The goal of these docs is simple: you should be able to understand what Fallback does, set it up, and use it confidently without needing to be a Git expert.
